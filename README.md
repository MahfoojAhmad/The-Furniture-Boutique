# The Furniture Boutique Website

A modern, responsive furniture e-commerce website with dynamic features and admin panel.

## Features

- **Fully Responsive Design**: Optimized for all devices using Bootstrap
- **Smooth Animations**: Scroll-triggered fade-in and slide-in animations
- **Blog System**: Dynamic blog with admin panel for content management
- **Interactive Elements**: Newsletter signup, contact forms, social media links
- **Product Showcase**: Carousel display of furniture items

## New Additions

### Blog Page (`blog.html`)
- Displays blog posts stored in localStorage
- Responsive card layout
- Admin access button

### Admin Panel (`admin.html`)
- Add new blog posts with title, image, and content
- Edit and delete existing posts
- Simple interface for content management

### Enhanced Animations
- Fade-in effects on scroll
- Hover animations on product cards
- Smooth transitions throughout

### Newsletter Signup
- Email subscription form in the footer
- Basic form validation and feedback

## File Structure

- `index.html` - Main homepage
- `blog.html` - Blog page
- `admin.html` - Admin panel
- `products.html` - Product listings
- `bed.html`, `dining.html`, `sofa.html` - Category pages
- `css/custom/style.css` - Custom styles
- `JS/script.js` - Main JavaScript functionality

## Technologies Used

- HTML5
- CSS3 with custom properties
- Bootstrap 5
- JavaScript (ES6+)
- jQuery
- Owl Carousel
- Font Awesome
- Google Fonts

## Getting Started

1. Open `index.html` in a web browser
2. Navigate to the Blog page to view posts
3. Access the Admin panel via the floating button on the blog page
4. Add sample posts through the admin interface

## Admin Panel Usage

1. Go to the blog page and click the admin button (cog icon)
2. Enter the admin password: `admin123` (you can change this in the code)
3. Upload images directly or add posts without images
4. Edit and delete existing posts
5. View posts on the blog page with full content in modal

## Blog Features

- **Image Upload**: Direct file upload instead of URL input
- **Full Content View**: Click "Read More" to see complete post in a modal
- **Admin Authentication**: Password-protected admin access
- **Responsive Design**: Works on all devices
- **Consistent Header/Footer**: Same navigation and footer as main site

## Security Note

The admin password is currently set to `admin123`. For production use, consider implementing proper authentication or changing the password in the code.
